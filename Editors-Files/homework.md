# M3 Homework - Editors, Files, Streams

## Steps

1. Copied /etc/services to local file
2. Extracted comment lines using grep
3. Removed comments using grep -v
4. Filtered UDP services
5. Edited file using vi (deleted from specific line to end)
6. Replaced / with - using sed
7. Created doc1 and doc2 using heredoc
8. Combined files using paste
9. Extracted fields using cut
10. Found unique values using sort and uniq
11. Counted occurrences
12. Found small files in /etc

## Commands used

grep, sed, cut, paste, find, vi

## Notes

Practical exercise for Linux text processing and file manipulation.
